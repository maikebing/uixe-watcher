# Uixe.Copilot Web

基于 `README.md` 中的前端规划新增的 Web UI 骨架。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Arco Design Vue
- Tailwind CSS
- ECharts
- SignalR Client

## 当前页面

- 监控总览
- 收费站监控
- 事件中心
- 事件详情
- 历史查询
- 系统配置

## 说明

当前为第一版 UI 骨架，使用本地 mock 数据展示深色监控风格布局，后续将接入 `Uixe.Copilot.Api` 与 SignalR。
